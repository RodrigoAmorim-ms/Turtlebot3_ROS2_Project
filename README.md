# TurtleBot3 ROS 2 Mapping Project

A learning project using ROS 2, TurtleBot3, and Gazebo to:

1. map an unknown indoor environment
2. save the occupancy map
3. relocalize using AMCL
4. extend to autonomous waypoint patrol

## Planned stack

- Ubuntu 22.04 (WSL)
- ROS 2 Humble
- TurtleBot3
- Gazebo
- RViz
- Nav2
- AMCL
